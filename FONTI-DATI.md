# Fonti Dati

Questa pagina riassume le fonti che possiamo considerare per il vocabolario online.

## 1. Treccani

Uso consigliato: consultazione pubblica manuale.

Uso NON consigliato:

- copia del contenuto nel nostro database
- riproduzione sistematica delle definizioni
- creazione di un archivio derivato

Motivo:

- i contenuti sono protetti
- la riproduzione senza consenso scritto e' vietata

Riferimenti:

- https://www.treccani.it/footer/termine-condizioni/
- https://www.treccani.it/footer/condizioni/

## 2. Morph-it!

Uso consigliato:

- base del lemmario
- forme flesse
- informazioni morfologiche

Punti forti:

- gratuito
- molto utile per la lingua italiana
- adatto a costruire una base dati iniziale

Limite:

- non e' un vocabolario completo di definizioni editoriali

Licenza:

- CC BY-SA

Riferimento:

- https://docs.sslmit.unibo.it/doku.php?id=resources%3Amorph-it

## 3. Wikizionario

Uso consigliato:

- definizioni
- esempi
- locuzioni
- contenuti integrativi

Punti forti:

- contenuti aperti e riusabili secondo licenza
- struttura gia' orientata al dizionario

Limiti:

- qualita' non uniforme
- copertura variabile da voce a voce

Licenza:

- CC BY-SA

Riferimenti:

- https://en.wiktionary.org/wiki/Wiktionary%3ACopyrights
- https://en.wiktionary.org/wiki/Wiktionary%3ATerms_of_use

## 4. De Mauro online

Uso consigliato:

- consultazione
- confronto qualitativo

Uso non consigliato senza chiarimenti specifici:

- import automatico nel nostro database
- copia delle definizioni nel nostro prodotto

Riferimenti:

- https://dizionario.internazionale.it/
- https://www.internazionale.it/copyright
- https://www.internazionale.it/termini-e-condizioni

## 5. Scelta pratica per partire

La combinazione iniziale piu' utile e' questa:

- `Morph-it!` per la lista delle parole
- `Wikizionario` per le definizioni riusabili

## 6. Regola di buon senso

Ogni voce del vocabolario deve poter dire chiaramente:

- da dove arriva
- con quale licenza si puo' riusare
- se e' stata modificata o adattata
